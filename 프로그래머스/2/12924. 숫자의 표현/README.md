### 문제 설명

<p style="user-select: auto !important;">Finn은 요즘 수학공부에 빠져 있습니다. 수학 공부를 하던 Finn은 자연수 n을 연속한 자연수들로 표현 하는 방법이 여러개라는 사실을 알게 되었습니다. 예를들어 15는 다음과 같이 4가지로 표현 할 수 있습니다.</p>

<ul style="user-select: auto !important;">
<li style="user-select: auto !important;">1 + 2 + 3 + 4 + 5 = 15</li>
<li style="user-select: auto !important;">4 + 5 + 6 = 15</li>
<li style="user-select: auto !important;">7 + 8 = 15</li>
<li style="user-select: auto !important;">15 = 15</li>
</ul>

<p style="user-select: auto !important;">자연수 n이 매개변수로 주어질 때, 연속된 자연수들로 n을 표현하는 방법의 수를 return하는 solution를 완성해주세요.</p>
